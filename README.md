## Redux-bookstore

This is a bookstore project built using Redux.

## Built With

- React
- npm
- Node.js
- Redux
- ES6
- Webpack

## Linters used

- ESlint

## Live Demo

[Production version](https://redux-bookstr.herokuapp.com/)

## Clone the project

```
git clone https://github.com/onedebos/redux-bookstore.git
```

## Install dependencies

```
npm install
```

or

```
yarn install
```

## Run the app

```
yarn start
```

or

```
npm start
```
